接口

药材获取列表

根据版本获取药材列表，符合版本情况，则返回Redis缓存数据，否则更新缓存，并下放数据

  • 接口类型

    GET

  • 请求参数

字段 类型 名称 必填项 功能
AppVersion Int APP版本号 是 当前客户端版本
DataVersion Int 数据版本号 是 高等于缓存版本不下放数据，低于缓存版本返回数据
  • 返回字段
返回字段 字段类型 说明
status Int 100 = 请求成功
message String 描述
version Int 版本号

  • 返回事例

    不下放数据

    下放数据
