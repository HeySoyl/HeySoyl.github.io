    编译，注意cmake \后面操作都带有“-”符号

    编译成功，执行安装

    MySQL5.7版本需增加“-DDOWNLOAD_BOOST=1 -DWITH_BOOST=/usr/local/boost”

    如果编译出现问题，执行“rm CMakeCache.txt”，便可重新编译

    创建用户

    有时候执行service mysql start命令报错env: /etc/init.d/mysql:权限不够问题
    解决办法：执行下面的语句
    chmod a+wrx /etc/init.d/mysql
