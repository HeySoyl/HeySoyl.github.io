命令

  1. compose以守护进程模式运行加-d选项

    docker-compose up -d

  2. 查看有哪些服务

    docker-compose ps

  3. 查看compose日志

    docker-compose logs web

    docker-compose restart

  6. kill compose服务

    docker-compose ps
    docker-compose kill

    docker-compose rm
