命令

  1. compose以守护进程模式运行加-d选项

    docker-compose up -d

  2. 查看有哪些服务

    docker-compose ps

  3. 查看compose日志

    docker-compose logs web
    docker-compose logs redis

  4. 停止compose服务

    docker-compose stop
    docker-compose ps

  5. 重启compose服务

    docker-compose restart
    docker-compose ps

  6. kill compose服务

    docker-compose ps
    docker-compose kill

  7. 删除compose服务

    docker-compose rm
